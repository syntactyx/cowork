# Cowork

A Claude AI assistant embedded in VS Code. Ask questions, get code suggestions, and collaborate without leaving your editor.

## Features

- Chat with Claude directly in VS Code via the Cowork panel
- Use `@cowork` in Copilot Chat for inline assistance
- Multiple conversations with persistent history
- Streaming responses with syntax highlighted markdown
- Insert code directly into your editor
- Smart paste — automatically wraps code in code blocks

## Usage

- Press `Ctrl+Shift+C` to open the Cowork panel
- Or type `@cowork` in Copilot Chat

## Requirements

An Anthropic API key stored in a `.env` file in the extension directory.
